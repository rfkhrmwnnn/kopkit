# Kopkit - Koperasi Kita Digital

A premium, modern, and responsive e-commerce web application for Kopkit built with Vue 3, Tailwind CSS (v4), and Pinia.

## Features

- **Splash Screen**: Animated introduction with a premium feel.
- **Authentication**: Login and Register (Mock) with Demo and Admin access.
- **Dashboard**: Auto-sliding promo carousel and featured products.
- **Shop**: Product search, category filtering, and unlimited cart addition.
- **Cart**: Edit quantities, remove items, and order summary.
- **Checkout**: Shipping address confirmation and payment method selection (Cash, Transfer, QRIS).
- **Profile**: Order history with status tracking (Packed, Shipped, Completed) and address management.
- **Admin Dashboard**: Comprehensive super admin panel with statistics, recent orders, and responsive sidebar.

## Getting Started

1. **Install Dependencies** (if not already done):

   ```bash
   npm install
   ```

2. **Run Development Server**:

   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Tech Stack

- Frontend: Vue 3 (Composition API)
- Styling: Tailwind CSS v4
- State Management: Pinia
- Routing: Vue Router
- Icons: Lucide Vue Next
- Tooling: Vite
